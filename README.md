Финальный проект по курсу "Аналитик данных"

Здесь моя задача была проверить, был ли успешен A/B тест по введению новой цены на премиум-подписку через 2 новые платёжные системы в некоторых странах на основе готового датасета. 

Было дано:
  1) Данные о пользователях:
      - в тестовой группе
      - в контрольной группе №1
      - в контрольной группе №2
  2) Данные о транзакциях:
      - в тестовой группе
      - в контрольной группе №1
      - в контрольной группе №2

Стек: python, pandas, numpy, pingouin, scipy, seaborn, matplotlib

A/B тест проведен с применением методов статистического анализа, таких как т-тест Стьюдента, т-тест Уэлча, бутстрап разниц средних величин и отдельно сумм, критерий хи-квадрат
